# object_B: 文本到 3D 资产生成

物体 B 是使用 threestudio + DreamFusion 从文本 prompt 生成的篮球资产。

## 内容

- `configs/`: 训练/续训命令和配置。
- `results/save/`: DreamFusion 输出结果，包括训练过程渲染图、`it6000-test.mp4`、测试帧和 `it6000-export/` mesh。
- `results_preview/`: 最终篮球渲染预览图。
- `external_repos/threestudio/`: 第三方 threestudio 仓库占位目录，源码未复制。
